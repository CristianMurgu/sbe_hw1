Generator de seturi echilibrate de publicații și subscripții cu suport pentru paralelizare și salvare în fișiere text.

## Cerințe Sistem
- Java 11 sau mai nou
- Maven 3.6+
- Sistem de operare: Windows/Linux/macOS
